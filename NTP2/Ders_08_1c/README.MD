# Ders_08_1c

## Proje Temasi
Varlik, insan ve bebek modeliyle hiyerarsik kalitim.

## Proje Turu
Konsol uygulamasi.

## Kullanilan OOP Kavramlari
Cok seviyeli kalitim, base class davranisi ve ozellik devralma.

Bu projede kodun ana fikri, veriyi ve davranisi ilgili siniflar icinde toplamak ve uygulama akisinda bu nesneleri kullanmaktir. Form, web veya konsol yapisina gore kullanici etkilesimi ayri bir katmanda tutulurken model siniflari uygulamanin temel kavramlarini temsil eder.

## Onemli Dosyalar
- `Ders_08_1c\Models\Bebek.cs`
- `Ders_08_1c\Models\Insan.cs`
- `Ders_08_1c\Models\Varlik.cs`
- `Ders_08_1c\Program.cs`

dotnet run --project "Ders_08_1c\Ders_08_1c.csproj"
