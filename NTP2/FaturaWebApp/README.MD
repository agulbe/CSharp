# FaturaWebApp

## Proje Temasi
ASP.NET Core Razor Pages ile fatura sayfasi temelli web uygulamasi.

## Proje Turu
ASP.NET Core web projesi.

## Kullanilan OOP Kavramlari
Razor Pages, page model, MVC benzeri ayrim, routing ve web uygulamasi yasam dongusu.

Bu projede kodun ana fikri, veriyi ve davranisi ilgili siniflar icinde toplamak ve uygulama akisinda bu nesneleri kullanmaktir. Form, web veya konsol yapisina gore kullanici etkilesimi ayri bir katmanda tutulurken model siniflari uygulamanin temel kavramlarini temsil eder.

## Onemli Dosyalar
- `FaturaWebApp\Pages\Error.cshtml.cs`
- `FaturaWebApp\Pages\Index.cshtml.cs`
- `FaturaWebApp\Pages\Privacy.cshtml.cs`
- `FaturaWebApp\Program.cs`

## Calistirma
```bash
dotnet run --project "FaturaWebApp\FaturaWebApp.csproj"
```
