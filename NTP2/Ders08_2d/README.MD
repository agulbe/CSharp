# Ders08_2d

## Proje Temasi
Insan ve evcil hayvan modelleriyle buyume davranisinin ortaklastirilmasi.

## Proje Turu
Konsol uygulamasi.

## Kullanilan OOP Kavramlari
Interface, polimorfizm, soyutlama ve farkli siniflarda ayni davranis sozlesmesi.

Bu projede kodun ana fikri, veriyi ve davranisi ilgili siniflar icinde toplamak ve uygulama akisinda bu nesneleri kullanmaktir. Form, web veya konsol yapisina gore kullanici etkilesimi ayri bir katmanda tutulurken model siniflari uygulamanin temel kavramlarini temsil eder.

## Onemli Dosyalar
- `Ders08_2d\Model\EvcilHayvan.cs`
- `Ders08_2d\Model\IBuyume.cs`
- `Ders08_2d\Model\Insan.cs`
- `Ders08_2d\Program.cs`

dotnet run --project "Ders08_2d\Ders08_2d.csproj"
