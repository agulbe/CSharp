# WinFormsApp2

## Proje Temasi
Ogrenci sinifi ve Windows Forms arayuzu ile veri gosterimi.

## Proje Turu
Windows Forms uygulamasi.

## Kullanilan OOP Kavramlari
Model sinifi, kapsulleme, form olaylari ve UI ile nesne baglantisi.

Bu projede kodun ana fikri, veriyi ve davranisi ilgili siniflar icinde toplamak ve uygulama akisinda bu nesneleri kullanmaktir. Form, web veya konsol yapisina gore kullanici etkilesimi ayri bir katmanda tutulurken model siniflari uygulamanin temel kavramlarini temsil eder.

## Onemli Dosyalar
- `WinFormsApp2\Form1.cs`
- `WinFormsApp2\Form1.Designer.cs`
- `WinFormsApp2\Ogrenci.cs`
- `WinFormsApp2\Program.cs`

## Calistirma
```bash
dotnet run --project "WinFormsApp2\WinFormsApp2.csproj"
```
