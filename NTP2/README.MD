# NTP2 2026 C# Projeleri

Bu klasor, `D:\Code\NTP2\2026` altindaki NTP2 ders projelerinin `agulbe/CSharp` reposu icin duzenlenmis aktarimidir.

`GP!` ve `GP^` ile baslayan proje klasorleri bu aktarima dahil edilmemistir. Kaynakta bu adlarla eslesen klasor bulunmadigi icin mevcut NTP2 projeleri kopyalanmistir. Derleme ciktisi olan `bin`, `obj`, `.vs` ve kullaniciya ozel Visual Studio dosyalari aktarilmamistir.

## Genel Tema
Projeler C# ve .NET ekosisteminde nesne yonelimli programlama, Windows Forms, ASP.NET Core Razor Pages/MVC, Entity Framework ve .NET MAUI konularini ders ornekleri uzerinden gostermektedir.
