# MyStore

## Proje Temasi
Razor Pages ile musteri kaydi tutan magazacilik uygulamasi.

## Proje Turu
ASP.NET Core web projesi.

## Kullanilan OOP Kavramlari
PageModel, model binding, form isleme, kapsulleme ve web sayfasi organizasyonu.

Bu projede kodun ana fikri, veriyi ve davranisi ilgili siniflar icinde toplamak ve uygulama akisinda bu nesneleri kullanmaktir. Form, web veya konsol yapisina gore kullanici etkilesimi ayri bir katmanda tutulurken model siniflari uygulamanin temel kavramlarini temsil eder.

## Onemli Dosyalar
- `MyStore\Pages\Clients\Create.cshtml.cs`
- `MyStore\Pages\Clients\Index.cshtml.cs`
- `MyStore\Pages\Error.cshtml.cs`
- `MyStore\Pages\Index.cshtml.cs`
- `MyStore\Pages\Privacy.cshtml.cs`
- `MyStore\Program.cs`

## Calistirma
```bash
dotnet run --project "MyStore\MyStore.csproj"
```
