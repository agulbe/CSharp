# GorevApp

Bu proje, .NET MAUI ile geliştirilmiş SQLite tabanlı görev takip uygulamasıdır.

## Görev

- Görevleri listelemek.
- Yeni görev eklemek ve mevcut görevi düzenlemek.
- Görev başlığı, kategori, son tarih ve tamamlanma bilgisini modellemek.
- Görevleri yerel SQLite veritabanında saklamak.
- Kaydırma hareketiyle düzenleme ve silme işlemlerini desteklemek.

## Teknolojiler

- C#
- .NET MAUI
- XAML
- sqlite-net-pcl
- SQLite
- Dependency injection
