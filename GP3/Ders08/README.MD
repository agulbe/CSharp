# Ders08

Bu proje, resim açma, sayfa ayarları, yazıcı seçimi ve baskı önizleme işlemlerini gösterir.

## Görev

- Resim dosyası seçip form üzerinde göstermek.
- Sayfa ayarları iletişim kutusu ile kenar boşluğu, kağıt ve yön ayarlarını düzenlemek.
- Yazıcı seçimi ve baskı önizleme pencerelerini kullanmak.
- Seçilen resmi ve örnek metni yazdırmak.

## Teknolojiler

- C#
- .NET Windows Forms
- `OpenFileDialog`
- `PageSetupDialog`
- `PrintDialog`
- `PrintPreviewDialog`
- `PrintDocument`
