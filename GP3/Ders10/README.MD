# Ders10

Bu proje, özel bir switch kontrolünün forma eklenmesini gösterir.

## Görev

- `CheckBox` tabanlı `Switch` kontrolü oluşturmak.
- Açık ve kapalı durumlar için farklı renklerle özel çizim yapmak.
- Yuvarlatılmış gövde ve hareketli düğme görünümü çizmek.

## Teknolojiler

- C#
- .NET Windows Forms
- GDI+ / `System.Drawing.Drawing2D`
