# Ders03

Bu proje, yazıcı ayarlarını görüntüleme ve örnek metin yazdırma uygulamasıdır.

## Görev

- Sistemdeki yazıcıları listelemek ve varsayılan yazıcıyı seçmek.
- Seçili yazıcının kağıt, baskı kalitesi, renk desteği, geçerlilik, harmanlama ve dubleks bilgilerini göstermek.
- `PrintDocument` ile şiir metni içeren örnek bir çıktı üretmek.

## Teknolojiler

- C#
- .NET Windows Forms
- `System.Drawing.Printing`
