# Ders15

Bu proje, girilen sayıyı rakam görselleriyle ekranda gösterir.

## Görev

- `NumericUpDown` değerini okumak.
- Sayının her basamağını ilgili rakam görseliyle eşleştirmek.
- Rakamları form üzerinde dinamik `PictureBox` kontrolleriyle göstermek.
- Önceki rakam görsellerini temizleyip yeni sonucu oluşturmak.

## Not

Kodda `D:\Resim\Rakam\*.png` yolları kullanılıyor. Rakam görsellerinin yüklenebilmesi için bu dosyaların mevcut olması gerekir.

## Teknolojiler

- C#
- .NET Windows Forms
- Dinamik `PictureBox` oluşturma
