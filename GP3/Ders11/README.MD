# Ders11

Bu proje, metin dosyası açma ve sürükle-bırak ile dosya içeriği görüntüleme uygulamasıdır.

## Görev

- Dosya açma penceresiyle metin, web ve kaynak kod dosyalarını seçmek.
- Seçilen dosyanın yolunu ve içeriğini formda göstermek.
- Desteklenen uzantıdaki dosyaları `TextBox` üzerine sürükleyip bırakarak okumak.

## Teknolojiler

- C#
- .NET Windows Forms
- `OpenFileDialog`
- Drag and drop
