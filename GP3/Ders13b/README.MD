# Ders13b

Bu proje, form yüklendiğinde kod ile buton ve etiket oluşturma örneğidir.

## Görev

- Çalışma zamanında bir `Button` oluşturup forma eklemek.
- Butona tıklama olayı bağlamak.
- Tıklama sayısını dinamik oluşturulan `Label` üzerinde göstermek.
- Butonun renk, düz stil ve fare durumlarını kod ile ayarlamak.

## Teknolojiler

- C#
- .NET Windows Forms
- Dinamik kontrol oluşturma
- Olay bağlama
