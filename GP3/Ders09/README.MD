# Ders09

Bu proje, özel bir aç/kapat anahtarı kontrolü geliştirme denemesidir.

## Görev

- `CheckBox` sınıfından türeyen `Anahtar` adlı özel kontrol oluşturmak.
- Açık/kapalı durumlar için farklı arka plan ve düğme renkleri tanımlamak.
- Yuvarlatılmış anahtar gövdesi için `GraphicsPath` hazırlamak.

## Not

Form üzerinde ek davranış bulunmuyor; özel kontrol sınıfı başlangıç aşamasındadır.

## Teknolojiler

- C#
- .NET Windows Forms
- GDI+ / `System.Drawing.Drawing2D`
