# Ders12

Bu proje, çalışma zamanında dinamik Windows Forms kontrolleri oluşturma örneğidir.

## Görev

- Kullanıcının seçtiği öğrenci sayısına göre form üzerinde satırlar oluşturmak.
- Her öğrenci için numara etiketi, ad-soyad kutusu ve vize notu girişi eklemek.
- `Label`, `TextBox` ve `NumericUpDown` kontrollerini kod ile üretmek.

## Teknolojiler

- C#
- .NET Windows Forms
- Dinamik kontrol oluşturma
