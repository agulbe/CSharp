# Ders01a

Bu proje, `Ders01` ile aynı yazdırma örneğinin ikinci kopyasıdır.

## Görev

- Sistemdeki yazıcıları açılır kutuda göstermek.
- Seçilen yazıcıya basit bir metin çıktısı göndermek.
- Yazıcı seçilmezse kullanıcıyı uyarı mesajı ile bilgilendirmek.

## Teknolojiler

- C#
- .NET Windows Forms
- `System.Drawing.Printing`
