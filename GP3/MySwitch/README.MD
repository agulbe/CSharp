# MySwitch

Bu proje, iki özel Windows Forms kontrolünü gösteren örnek uygulamadır.

## Görev

- `ModernSwitch` adlı metinli aç/kapat kontrolü oluşturmak.
- `StatusButton` adlı yuvarlatılmış, durum göstergeli buton oluşturmak.
- Switch durumuna göre butonu etkinleştirmek veya devre dışı bırakmak.
- Buton üzerinde açık/kapalı durumunu renkli göstergeyle belirtmek.

## Teknolojiler

- C#
- .NET Windows Forms
- Özel kontrol geliştirme
- GDI+ / `System.Drawing.Drawing2D`
