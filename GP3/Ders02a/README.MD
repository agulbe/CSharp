# Ders02a

Bu proje, `Ders02` ile aynı yazıcı özellikleri örneğinin ikinci kopyasıdır.

## Görev

- Yazıcı listesini almak.
- Varsayılan yazıcıyı bulup seçmek.
- Yazıcı değiştiğinde kağıt boyutlarını ve baskı kalitelerini güncellemek.

## Teknolojiler

- C#
- .NET Windows Forms
- `System.Drawing.Printing`
