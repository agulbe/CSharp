# Ders04

Bu proje, Windows Forms üzerinde grafik çizim ve aynı çizimi yazdırma örneğidir.

## Görev

- Form üzerinde resim, elips, dikdörtgen, yay, Bezier eğrisi, çizgi, pasta dilimi, çokgen ve yuvarlatılmış dikdörtgen çizmek.
- Bazı şekilleri dolgu fırçalarıyla boyamak.
- Form tıklanınca aynı grafik içeriğini yazıcıya göndermek.
