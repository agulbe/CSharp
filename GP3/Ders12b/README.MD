# Ders12b

Bu proje, SQL Server'dan öğrenci verisi okuyup forma dinamik kontrol olarak yerleştirme örneğidir.

## Görev

- `Okul` veritabanındaki `Ogrenci` tablosundan öğrenci kayıtlarını okumak.
- Her öğrenci için numara, ad-soyad ve vize notu alanlarını formda dinamik oluşturmak.
- Öğrenci verisini `Ogrenci` modeli ile temsil etmek.

## Not

Bağlantı dizesi yerel SQL Server için ayarlanmıştır: `(local)`, `Okul` veritabanı ve `Ogrenci` tablosu gerekir.

## Teknolojiler

- C#
- .NET Windows Forms
- Microsoft.Data.SqlClient
- SQL Server
