# Ders02c

## Görev

Bu proje, kullanıcı adı doğrulaması yapan ve buton olaylarıyla etkileşimli davranış gösteren selamlama uygulamasıdır.

## İşlenen konular

- `Form.Load` olayında buton özelliklerini çalışma zamanında ayarlama.
- Boş `TextBox` kontrolü için kullanıcıyı uyarma.
- Butonun fare üzerine gelince büyümesi, ayrılınca eski boyutuna dönmesi.
- Buton boyutu değiştiğinde `Label` üzerinde durum bilgisi gösterme.

## Kullanılan kontroller

- `Label` (`label1`, `lblDurum`)
- `TextBox` (`txtAd`)
- `Button` (`btnSelam`, `btnKapat`)
