# Ders11c

## Görev

Bu proje, `LinkLabel` kontrolleri ve bağlantı tıklama olaylarını gösterir.

## İşlenen konular

- `LinkLabel.LinkClicked` olayını kullanma.
- Bağlantıya tıklandığında kullanıcıya mesaj gösterme.
- `LinkVisited` özelliğiyle bağlantının ziyaret edildiğini işaretleme.
- `ImageList` ile bağlantı/görsel kaynakları için hazırlık.

## Kullanılan kontroller

- `LinkLabel` (`linkLabel1`, `linkLabel2`, `linkLabel3`, `linkLabel4`)
- `ImageList` (`imgListe`)
