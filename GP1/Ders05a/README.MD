# Ders05a

## Görev

Bu proje, `ListBox` üzerinde seçilen öğeyi algılama ve sonucu etikete yazma örneğidir.

## İşlenen konular

- `ListBox.SelectedIndexChanged` olayını kullanma.
- Seçim yapılmışsa `SelectedItem` değerini okuma.
- Seçilen öğeyi `Label` üzerinde gösterme.
- `SelectionMode` özelliğiyle çoklu seçim denemesi için hazırlık.

## Kullanılan kontroller

- `Label` (`label1`, `label2`, `label3`)
- `ListBox` (`lstAdlar`)
