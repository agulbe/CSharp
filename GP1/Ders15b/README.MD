# Ders15b

## Görev

Bu proje, taksit tarihlerini hesaplayıp listeleyen basit tarih uygulamasıdır.

## İşlenen konular

- `NumericUpDown` ile taksit sayısı ve ayın günü değerlerini alma.
- `DateTime.Now.AddDays` ile gelecekteki tarihleri hesaplama.
- Hesaplanan tarihleri `ListBox` içine ekleme.

## Kullanılan kontroller

- `ListBox` (`lstTaksit`)
- `Label` (`label1`, `label2`)
- `NumericUpDown` (`nudTaksit`, `nudGun`)
- `Button` (`btnHesapla`)
