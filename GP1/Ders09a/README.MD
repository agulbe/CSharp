# Ders09a

## Görev

Bu proje, onay penceresi ve `DialogResult` kullanımını gösterir.

## İşlenen konular

- `MessageBox.Show` ile `YesNoCancel` düğmeleri içeren soru penceresi açma.
- Kullanıcının bastığı düğmeyi `DialogResult` ile kontrol etme.
- `Yes` seçilirse formu kapatma, diğer seçimlerde `Label` metnini güncelleme.

## Kullanılan kontroller

- `Button` (`button1`)
- `Label` (`lblDugme`)
