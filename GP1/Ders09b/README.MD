# Ders09b

## Görev

Bu proje, `Timer` ile basit metin animasyonu yapmayı gösterir.

## İşlenen konular

- Animasyon karelerini string dizisinde tutma.
- `Form.Load` olayında zamanlayıcıyı başlatma.
- `Timer.Tick` ile `Label` metnini sırayla değiştirme.
- Etikete tıklanınca animasyonu durdurup formu kapatma.

## Kullanılan kontroller

- `Label` (`lblMetin`)
- `Timer` (`tmrAni`)
