# Ders12a

## Görev

Bu proje, dörtlük not ortalamasını yüzlük sisteme çeviren basit hesaplama uygulamasıdır.

## İşlenen konular

- `NumericUpDown.Value` değerini `double` olarak kullanma.
- Dörtlük-yüzlük dönüşüm formülü uygulama.
- Sonucu `MessageBox` ile kullanıcıya gösterme.

## Kullanılan kontroller

- `Label` (`label1`)
- `NumericUpDown` (`nudGano`)
- `Button` (`button1`)
