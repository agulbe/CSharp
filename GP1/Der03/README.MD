# Der03

## Görev

Bu proje, buton tıklama olayları ve `Timer` kullanımıyla form üzerindeki metni zaman içinde güncellemeyi gösterir.

## İşlenen konular

- `Button.Click` olayı ile mesaj kutusu gösterme.
- `Form.Load` olayında zamanlayıcı başlatma.
- `Timer.Tick` olayında buton metnini güncel tarih-saat ile değiştirme.

## Kullanılan kontroller

- `Button` (`button1`)
- `Timer` (`timer1`)
