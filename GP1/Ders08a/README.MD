# Ders08a

## Görev

Bu proje, tam ekran bir "balığı yakala" oyunu örneğidir.

## İşlenen konular

- Formu çalışma alanı boyutuna göre büyütme.
- `Random` ile butonu rastgele konumlandırma.
- `Timer.Tick` olayında butonu sürekli hareket ettirme.
- `MouseMove` ile fare koordinatlarını izleme.
- Butona tıklanınca geçen süreyi hesaplayıp yeniden başlatma seçeneği sunma.

## Kullanılan kontroller

- `Button` (`btnBalik`)
- `Timer` (`timer1`)
