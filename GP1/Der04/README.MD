# Der04

## Görev

Bu proje, `TextBox` içeriğini okuma, metin değişimini yakalama ve butonla mesaj gösterme örneğidir.

## İşlenen konular

- `TextBox.Lines` üzerinden son satırı okuma.
- `TextBox.TextChanged` olayında metin uzunluğunu kontrol etme.
- `MessageBox.Show` ile kullanıcıya bilgi gösterme.
- Tasarımda görsel kaynak listesi (`ImageList`) kullanımı.

## Kullanılan kontroller

- `Label` (`label1`)
- `TextBox` (`textBox1`)
- `Button` (`button1`, `button2`)
- `ImageList` (`imageList`)
